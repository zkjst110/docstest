Operators&Funtions 使用说明¶

说明¶

KVSQL是一款SQL合约执行引擎，可以运行符合范围的SQL语句，目标支持了部分常见运算符以及函数功能。以及在区块链的特殊场景下，并非所有功能都能够正常使用，部分函数被KVSQL限制以保障SQL智能合约执行的沙箱环境的安全。

限制功能主要包括如下：

  • 随机数

  • 部分浮点数函数
